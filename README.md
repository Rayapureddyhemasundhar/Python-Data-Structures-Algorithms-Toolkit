# Python Data Structures & Algorithms Toolkit

A beginner-friendly project that implements **fundamental Data Structures and Algorithms in Python from scratch**.
The main objective of this project is to understand **how core data structures work internally, how algorithms manipulate data, and how their efficiency can be analyzed using time and space complexity.**

This repository serves as a **learning resource for students and developers** who want to build a strong foundation in **Data Structures and Algorithms (DSA)**.

---

# Project Overview

Data Structures and Algorithms are essential concepts in computer science used to **store, organize, and process data efficiently**. Efficient data handling and algorithm design help improve program performance and scalability. ([GitHub][1])

In this project, we implemented commonly used **data structures and sorting algorithms without relying on built-in shortcuts**, allowing us to understand the **internal working mechanisms of each structure and algorithm**.

# Data Structures Implemented

The project includes the implementation of the following fundamental data structures:

### Stack

* Follows **LIFO (Last In First Out)** principle
* Supports operations like:

  * push
  * pop
  * peek
  * isEmpty

### Queue

* Follows **FIFO (First In First Out)** principle
* Supports operations like:

  * enqueue
  * dequeue
  * front
  * isEmpty

### Linked List

* Dynamic data structure consisting of nodes
* Each node contains:

  * data
  * reference to the next node

### Trees

* Hierarchical data structure
* Consists of nodes connected in parent-child relationships
* Used for structured data representation

### Graphs

* Collection of **vertices and edges**
* Useful for representing **networks and relationships**

---

# Algorithms Implemented

The project also includes implementations of common **sorting algorithms**.

### Quick Sort

A divide-and-conquer algorithm that selects a pivot element and partitions the array around it.

### Merge Sort

Another divide-and-conquer algorithm that divides the array into halves, sorts them, and merges them.

# Complexity Analysis

For each implemented data structure and algorithm, the following analysis was performed:

* **Time Complexity**
* **Space Complexity**
* **Best Case**
* **Average Case**
* **Worst Case**

This helps understand how the algorithm performs with increasing input size.

# Approach Used to Solve the Project

The project was developed using a **step-by-step learning approach**:

### Step 1 — Understanding the Concept

Before implementation, the theoretical concepts of each data structure and algorithm were studied.

### Step 2 — Designing the Structure

Each data structure was designed using **Python classes and methods**.

### Step 3 — Implementing from Scratch

All operations were implemented manually without relying on built-in implementations.

### Step 4 — Testing Operations

Each structure was tested using sample inputs to ensure correctness.

### Step 5 — Complexity Analysis

The performance of algorithms was evaluated using **time and space complexity analysis**.

### Step 6 — Demonstrations

Interactive examples were used to show how the data structures and algorithms work.
